# Doppelpunkt

Installation
============

Der Download erfolgt über den Butten "Code" oben rechts, wobei der Zip-Ordner des Programms heruntergeladen wird.

Nach dem Entpacken des ZIP-Ordners kann mit Anklicken von **setup.bat** die Installation ausgeführt werden. 

Mach der Installation können der heruntergeladene ZIP-Ordner und die entpackte Version davon natürlich gelöscht werden, denn bei der Installation werden die Dateien automatisch in den Unterordner "doppelpunkt" im Hauptordner des angemeldeten Windows-Benutzers bzw. in das Autostart-Menü installiert.

Dann müsste es funktionieren, mit STRG-B.
