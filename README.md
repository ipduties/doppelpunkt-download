# Doppelpunkt

Installation
============

Der Download erfolgt über den Butten "Code" oben rechts, wobei derZip-Ordner des Programms heruntergeladen wird.

Nach dem Entpacken des ZIP-Ordners kann mit Anklicken von **setup.bat** die Installation ausgeführt werden, wobei die Dateien automatisch in den Unterordner "doppelpunkt" im Hauptordner des angemeldeten Windows-Benutzers installiert wird.

Dann müsste es funktionieren, mit STRG-B.
